Объекты:
========
1. Примитивные типы данных передаются по значению.
    - number, string, boolean, null, undefined, symbol, bigint
2. Объектные типы данных передаются по ссылке!
    - object

Массивы:
========
Массив - это упорядоченный набор элементов каждый из которых имеет свой порядковый номер (индекс).

В JS индекс начинается с 0!

1. Длинна массива = количество элементов в массиве
2. Sparse-array (Массив с "дырками").
3. Обход массива
4. Методы вставки и удаления элементов в/из массива
